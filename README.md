### 主要实现
利用Android最新控件实现：
* 保持ViewPager中Fragment的状态。
* 快速绑定已有数据的TabLayout与ViewPager。
* 在Fragment中显示标题栏，并为该标题栏添加菜单。
* Fragment嵌套Fragment。
* 一行代码解决RecyclerView等视图的滚动冲突。
* 为Fragment中的头部视图(含图片)实现沉浸式。
* 最简单的沉浸式版本适配。
